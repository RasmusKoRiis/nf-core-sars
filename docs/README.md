# nf-core/sars: Documentation

The nf-core/sars documentation is split into the following pages:

- [Usage](usage.md)
  - An overview of how the pipeline works, how to run it and a description of all of the different command-line flags.
- [Output](output.md)
  - An overview of the different results produced by the pipeline and how to interpret them.
- [Offline cache preparation](offline-cache.md)
  - How to populate local resources required by wrapper offline mode.
- [Offline harddrive installation](offline-harddrive-install.md)
  - How to create a portable harddrive bundle and run it on an offline computer.

You can find a lot more documentation about installing, configuring and running nf-core pipelines on the website: [https://nf-co.re](https://nf-co.re)
